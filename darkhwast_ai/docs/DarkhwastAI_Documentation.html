1. Executive Summary

In Pakistan, bureaucratic complexities, opaque billing systems, and sudden tax regulations place an immense burden on the average citizen. Each month, millions of utility consumers receive overcharged bills, while taxpayers struggle to decode notices that contain hidden "ghost" deadlines. Despite the presence of redressing mechanisms, over 90% of grievances go unreported because citizens lack the legal expertise to draft formal complaints or identify specific regulatory violations.

DarkhwastAI is an autonomous end-to-end citizen rights enforcement agent that bridges this gap. By combining modern OCR, advanced multi-agent orchestration, and a localized database of Pakistani regulations, the application enables any citizen to upload a photo of a government document, instantly discover if their rights have been violated, receive a precise calculation of what they are owed, review a legally rigorous bilingual complaint, and simulate submission to the relevant redressing authority in under 90 seconds.

Core Value Proposition
"Apka document. Apka haq. 90 seconds." DarkhwastAI automates the transition from raw document ingestion to legally cited complaints, organizing group actions dynamically as clusters grow.

2. Technical Architecture & Agent Pipeline

DarkhwastAI implements a sequential multi-agent execution pipeline. The architecture uses a feature-first organization in Flutter and leverages **Riverpod** for robust state management. Each of the five specialized AI agents runs sequentially, communicating via typed Dart models to maintain strict state consistency and logical traceability.

3. Deep Dive into the 5 Specialized Agents

Agent 1: Document Intelligence Agent

Extracts all semantic data from raw OCR input text. The agent handles document classification into one of the four supported MVP categories: IESCO/WAPDA Bills, SNGPL/SSGC Gas Bills, BISP Letters, and FBR Income Tax Notices. It parses consumer reference numbers, billed amounts, billing dates, and payment deadlines.

Agent 2: Urgency Detector Agent

Scans the extracted text for obscure date patterns, hidden footnotes, stamp markers, and payment cutoffs. This agent acts as a safety valve: if a payment or response deadline is detected within 72 hours, it immediately interrupts the pipeline flow and generates an interactive, high-visibility "Pulsing Amber" warning on the user interface.

Agent 3: Rights Intelligence Agent

Cross-references the DocumentEntity with the Pakistani Law Knowledge Base seeded in Firestore. It extracts the approved regulatory rates for specific usage slabs and compares them directly against the customer's actual billed amount. It calculates the legal standing of the case as the **HAQ Score** (0-100), where ratings above 60 indicate highly robust standing, and details the exact refund amount owed to the citizen.

Agent 4: Action Drafter Agent

Synthesizes the legal violations, facts, and calculations to draft formal complaints. It generates two distinct versions of the complaint: a formal English complaint for filing and a Nastaliq-styled Urdu draft using native right-to-left formatting for the citizen's own understanding. It automatically inserts the calculated amounts owed and relevant regulatory references (e.g. Section 4(b) of the NEPRA Quarterly Tariff Adjustment May 2025).

Agent 5: Collective Pattern Agent

Performs a live query on Firestore to locate systemic patterns (same authority, same violation, same municipal area, and same month). If a threshold is crossed, it offers the citizen the choice to "Join Collective Action," compounding individual complaints into major group petitions. It also registers autonomous follow-ups in the Firestore queues to trigger reminders at 7, 14, and 30 days, escalating the case to the national ombudsman if ignored.

5. App UX Flows & Core Screens

DarkhwastAI utilizes a premium Pakistani-themed authority design system characterized by a Deep Teal primary theme (#0D3B44), Warm Amber accents (#F5A623), Off-White backgrounds, and clean, large rounded interfaces. The user flow is mapped across 8 premium screens:

  1. Splash & Onboarding: Multi-card walkthrough explaining document scanning, HAQ Score analysis, and automated filing.
  2. Home Screen: Displays aggregate citizen statistics, recent cases, and a prominent, pulsing amber capture button.
  3. Document Scanner: Complete device camera layout with overlay bounding boxes and real-time guidance prompts.
  4. Live Agent Trace: sequential vertical list representing the 5 agents. The system features typewriter animations, active glowing border pulses, and green checkmark animations.
  5. HAQ Dashboard: Animated 270° radial gauge that dynamically fills on load, accompanied by breakdown cards representing the amount owed, violation details, and legal precedents.
  6. Complaint Draft Viewer: Tabbed English/Urdu screens utilizing Noto Nastaliq Urdu fonts with RTL text alignment and a live editor option.
  7. Filing Confirmation: High-fidelity flight animations representing mock Citizens Portal submission, issuing a unique reference number (e.g. DW-2026-ISB-4521) and follow-up timeline blocks.
  8. Case Tracker: Vertical timeline status view (Filed → Under Review → Due → Resolved) with escalation warning overlays if resolution thresholds are breached.
Urdu Typography Standard
All Urdu content in the draft viewer and dashboards is displayed using the premium Noto Nastaliq Urdu font family, ensuring natural, beautiful script rendering for local users.

6. Local Demo Mode (Instructions for Judges)

To ensure perfect reliability during live evaluations and hackathon pitching, the application features an offline **Demo Mode** with pre-seeded scenarios that bypass external API networks entirely.

How to activate Demo Mode in the App:

  1. Go to the About ("Baare Mein") tab from the bottom navigation.
  2. Tap the Sparkle Icon logo (top center) exactly 5 times in rapid succession.
  3. A green notification pop-up will display: "Demo Mode ON — scripted mocks (for judges)".
  4. Select one of the pre-configured scenarios from the sliding chip menu:
    • IESCO Bill: Simulates a quarterly FCA overcharge, yielding a HAQ Score of 81 and a refund of Rs. 1,600, with 29 active collective cases.
    • FBR Tax (Urgent): Simulates an urgent tax notice showing a critical 4-day deadline, triggering the urgency pipeline interrupt and HAQ Score of 55.
    • SNGPL Gas Bill: Simulates meter reading variance overcharges.
    • BISP Letter: Simulates an appeal against a status rejection.
  5. Return to the Home Screen, select a document (or use the simulated scan option), and watch the 5-Agent pipeline execute seamlessly in real-time.
